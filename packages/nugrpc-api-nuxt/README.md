# NugRPC API Nuxt

> Nuxt module for NugRPC API

## Compatibilities

- Nuxt 3

## Installation

```bash
yarn add --dev @privyid/nugrpc-api-nuxt
```

Then, add to `nuxt.config.ts`

```ts
export default defineNuxtConfig({
  modules: ['@privyid/nugrpc-api-nuxt'],
})
```
